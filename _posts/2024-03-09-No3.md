  class Solution:
        def lengthOfLongestSubstring(self, s: str) -> int:
            visited = {}
            left = 0
            res = 0
            for i, c in enumerate(s):
                if c in visited and left <= visited[c]:
                    left = visited[c] + 1
                else:
                    res = max(res, i - left + 1)
                visited[c] = i
            return res

Given a string s, find the length of the longest 
substring
 without repeating characters.

**Example:**

**Input:** `s = "abcabcbb"`
**Output:** `3`
 
**Input:** `s = "bbbbb"`
**Output:** `1`

**Input:** `s = "pwwkew"`
**Output:** `3`
